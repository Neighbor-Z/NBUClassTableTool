# NBU课表工具 - SwiftUI

使用 SwiftUI 重写的NBU课表工具。

得益于采用原生开发库和强大的 SwiftUI 框架，macOS 版本开发更专注更高效，在实现相同核心功能的同时又带来基于 macOS 平台的独特体验。

## 特点

-支持深色模式的现代 SwiftUI 界面

-从 ehall 系统获取课程表

-生成 ICS 日历文件以导入到日历应用程序

-支持本科生和研究生

## 系统要求

- macOS 12.0 或更高版本

## 用法

1.输入学号

2.输入密码

3.输入学期，例如“2025-2026-1”

4.输入学期的第一个星期一（首个周一），例如“2025-09-08”

5.点击“获取”以试图获取您的课程表

6.选择一个位置来保存生成的 ICS 文件

## 技术说明

-使用 async/await 进行网络操作

-为密码验证实施 AES-CBC 加密

-处理登录表单提取 HTML 解析

-生成标准 ICS 日历格式文件
